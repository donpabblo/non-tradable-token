# Install
```
npm i
```

# Start local hardhat node
```
npx hardhat clean
npx hardhat node
```

# Setup and Run
On a different terminal:
```
npx hardhat compile
npx hardhat deploy
npm run start
```
Go to https://localhost:3000
Enjoy!

# Test Contract
```
npm run test
```
